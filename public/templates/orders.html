Welcome to the order screen.

  • To select a different table click on table select.

  • Click on a category item such as dinner to view dinner related items, then select an item to add to the current check. To go back to the categories, select main menu.

  • To add a check to the table, select add check or to remove an empty check select remove empty check.

  • On the right, you can click the arrows to view each check seperately and clicking on an item and selecting delete item will remove it from the check.

  • Selecting a specific check and clicking on checkout will bring up the checkout screen.
